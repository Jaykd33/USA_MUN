# 🎯 WAR ROOM — FOLDER 7: POI WAR ROOM
100 POIs, ranked Easy → Committee-Killer. Each includes: **Purpose** (why ask it) · **Expected Answer** (what the target will likely say) · **Follow-Up → Trap → Knockout** (the three-step sequence to close it out).

**How to use this folder:** Don't fire Committee-Killers early — they work because the room hasn't heard the underlying facts yet. Build with Easy/Medium POIs in your first moderated caucus to establish your command of the material, save Hard/Killer tier for when a specific delegate over-claims and hands you the opening.
